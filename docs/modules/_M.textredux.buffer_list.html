Module _M.textredux.buffer_list

The buffer list module provides a text based replacement for the standard TextAdept buffer list.

Two differences compared to the stock one is the ability to close a buffer directly from the buffer list (bound to Ctrl + d by default), and the option of specifying the buffers to list via a provided function.
