Modules

_M.textredux The Textredux module allows you to easily create text based interfaces for the Textadept editor and offers a set of text based interfaces.
_M.textredux.buffer_list The buffer list module provides a text based replacement for the standard Textadept buffer list.
_M.textredux.core The Textredux core module allows you to easily create text based interfaces for the Textadept editor.
_M.textredux.core.buffer The buffer class wraps a Textadept buffer, and extends it with support for custom styling, buffer specific key bindings and hotspot support.
_M.textredux.core.indicator The indicator module provides support for indicators in your buffers.
_M.textredux.core.list The list class provides a versatile and extensible text based item listing for Textadept, featuring advanced search capabilities and styling.
_M.textredux.core.style The style module lets you define and use custom, non-lexer-based styles.
_M.textredux.fs textredux.fs provides file system related functions and a text based file browser for Textadept.
_M.textredux.gui The gui module handles GUI related operations for Textredux.
_M.textredux.hijack The hijack module provides the easiest, and most invasive, way of getting Textredux functionality for Textadept.
